
mag commit kapag may nabago ka sa code or naadd sa project folder nato:
	-git add .
	-git commit -m "mga nabago mo lagay mo dito";

mag push if gusto mo nang i update yung project file sa github:
	-git push origin main

mag pull if may mga nabago or nadagdag sa project file sa github: (mag pull lang if gusto mo ma update ang file na nasa computer mo)
note na bago ka mag pull, i back up ang file na ginagawa or copy the line of code na ginawa tsaka mag pull then paste ang lines of code na ginawa sa bagong pull
	-git pull origin main
